## De opdracht 

Dit is de start-up project die ik gemaakt heb voor "Digital Natives". Bij dit project heb ik een campagne website gemaakt over web toegankelijkheid. De bedoeling van de opdracht is dat de opdracht toegankelijk is voor mensen met een screen reader maar niet voor mensen zonder disability's.
